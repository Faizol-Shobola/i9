
# Generate a CHIP-007 Format
Converts csv file that contains NFT details to JSON format and calculating the sha256 of each NFT

## Run Locally

Clone the project

```bash
  git clone https://github.com/Faizol-Shobola/i9.git
```

Go to the project directory

```bash
  cd NFT
```

Run python project

```bash
  python app.py
```

Input file name
```bash
hngi9.csv
```
